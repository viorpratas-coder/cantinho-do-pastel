# Instruções para o Sistema de Fidelidade por Carimbos

## Visão Geral

O sistema de fidelidade permite que os clientes coletem carimbos a cada compra e, após acumular 5 carimbos, ganhem um pastel grátis. Como os pagamentos são feitos manualmente via WhatsApp, é essencial que a equipe siga os procedimentos corretos para garantir que os clientes recebam seus carimbos corretamente.

## Fluxo de Funcionamento

1. **Cliente faz pedido via site**
   - O cliente acessa o site e faz seu pedido
   - O cliente acessa a página de fidelidade e gera um código único para o pedido
   - O código é automaticamente copiado para a área de transferência

2. **Cliente paga via WhatsApp**
   - O cliente envia o código único ao fazer o pagamento via WhatsApp
   - A equipe deve anotar o código em conjunto com o pedido

3. **Confirmação do pagamento**
   - A equipe acessa o painel administrativo em `/admin/fidelidade`
   - Insere o código do pedido recebido via WhatsApp
   - Confirma o pagamento e adiciona o carimbo ao cliente

## Procedimentos Detalhados

### 1. Recebendo Pagamentos no WhatsApp

Quando um cliente enviar uma mensagem para fazer um pedido:

1. Anote os itens solicitados
2. Informe o valor total do pedido
3. Peça ao cliente para acessar a página de fidelidade no site e gerar um código
4. Solicite que o cliente envie o código gerado
5. Anote o código junto com os itens do pedido

### 2. Confirmando Pagamentos e Adicionando Carimbos

Após receber o pagamento:

1. Acesse o painel administrativo: `seusite.com/admin/fidelidade`
2. Insira o código do pedido no campo apropriado
3. Verifique se:
   - O código está correto
   - O valor pago corresponde ao pedido
   - O pedido foi realmente entregue
4. Clique em "Confirmar Pagamento e Adicionar Carimbo"
5. O carimbo será automaticamente adicionado ao cliente

### 3. Resgatando Prêmios

Quando um cliente tiver 5 carimbos:

1. O cliente pode resgatar o prêmio diretamente no site
2. O cliente receberá um código exclusivo para o pastel grátis
3. O cliente deve apresentar esse código na próxima compra
4. A equipe deve verificar o código antes de liberar o pastel grátis

## Boas Práticas

### Importante:
- **Nunca** adicione carimbos sem confirmar o pagamento
- **Sempre** verifique se o código está correto antes de confirmar
- **Mantenha** um registro dos códigos utilizados para evitar fraudes
- **Instrua** os clientes sobre como gerar e usar os códigos

### Dicas:
- Explique o programa de fidelidade para os clientes no momento do pedido
- Incentive os clientes a acessarem a página de fidelidade para gerar códigos
- Verifique regularmente o painel administrativo para garantir que todos os pagamentos foram confirmados

## Solução de Problemas

### Cliente não consegue gerar código
- Verifique se o cliente está acessando o site corretamente
- Instrua o cliente a clicar no botão "Gerar Código do Pedido"
- O código deve aparecer no campo de texto e ser copiado automaticamente

### Código não reconhecido no painel administrativo
- Verifique se o código foi digitado corretamente (incluindo maiúsculas e minúsculas)
- Certifique-se de que o código não expirou (os códigos não expiram, mas é bom verificar)
- Se o problema persistir, entre em contato com o suporte técnico

### Cliente com 5 carimbos não consegue resgatar prêmio
- Verifique se o cliente está acessando a página de fidelidade corretamente
- Confirme se os 5 carimbos foram realmente adicionados
- Instrua o cliente a clicar no botão "Resgatar Pastel Grátis"
